## Usage

Make sure you have [Node.js](https://nodejs.org) installed.

Clone this repo and `cd` to the project directory.

```
npm i
```

Run the development server to preview locally at http://localhost:8080/.

```
npm start
```

Create a production build.

```
npm run build
```
